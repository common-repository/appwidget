This plugin lets you easy access to information on any Android application the appchina.com market.
 
== Description ==

With appwidget you can easily share your Android app on your blog posts.

For example if you need to promote the Appchina app ( Market link for Appchina ) which has com.yingyonghui.market as package name, you only need to insert com.google.android.youtube in your editor and it will generate:
* icon
* name
* developer
* rating
of the app.

Website:
http://i.appchina.com/wp-plugin/

== Installation ==
 
1. Put this under plugins folder (/wp-content/plugins).
2. Go to Admin->Plugins and activate the plugin "appwidget".
